Description
-----------
This is a minigolf game using the physics engine [jBox2d](http://www.jbox2d.org/)  
See the rules and controls in `HOW TO PLAY.txt`

Launch
------
On Linux, to compile and launch, enter `./LAUNCH.sh` in a terminal
